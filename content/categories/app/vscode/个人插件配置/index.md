# vscode 插件使用





## clangd--智能提示

以ubuntu为例，需要先在ubuntu中安装



~~~
sudo apt install llvm clangd lldb
~~~



再在打开vscode，下载两款插件 clangd  和 codelldb

![image-20220901150949828](images/image-20220901150949828.png)

其中CodeLLDB可能会下载不成功，可以从vsix安装
